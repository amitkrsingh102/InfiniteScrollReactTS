## Installation
### Steps:-
* clone the repository
    ```
    git clone https://github.com/amitkrsingh102/InfiniteScrollReactTS.git
    ```
* install the dependencies
    ```
    cd InfiniteScrollReactTS
    pnpm install
    ```
    or
    ```
    cd InfiniteScrollReactTS
    npm install
    ```
* run locally
    ```
    pnpm run dev
    ```
    or
    ```
    npm run dev
    ```
